# AI4PD_2025
Tutorials, Notebooks and Slides for the EMBO Practical Course: Latin American AI4PD workshop 2025 | Puerto Varas, Chile

## Slides
Slides can be found under [docs](./docs/), grouped by topic.

## Tutorials
Tutorials are meant to be "mini" design problems to demonstrate how to use particular pieces of software. Once you have run a particular tutorial, you may find it more helpful to visit its corresponding notebook for your particular design problem. Tutorials can be found [here](./tutorials/).

## Notebooks
The notebooks shared in this repo are designed to be run on Google Colab. Each notebook corresponds to a particular piece of software. Notebooks can be found [here](./notebooks/).
